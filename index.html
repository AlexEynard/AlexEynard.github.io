Présentation du site

Bienvenue sur ce site dédié aux mathématiques et à la physique.

Ce site propose des exercices variés avec une corrections détaillées, destinés à aider les étudiants à progresser. Le code source du site (HTML, CSS et JavaScript) est disponible sur GitHub, qui sert également à son hébergement. Pour l’instant, le site couvre un nombre limité de chapitres, principalement ceux étudiés dans le cadre de mes cours, mais il est en constante évolution.

Historique des modifications :

[14/12/2024] Ajout d'exercices de mécanique.

[06/11/2024] Ajout du chapitre sur les fonctions de plusieurs variables.

[18/09/2024] Ajout du chapitre sur les équations différentielles.

[12/09/2024] Mise en ligne du site.

Prochains ajouts :

De nouveaux exercices dans le chapitre des équations différentielles.

Ajout d'une section sur les applications en physique pour les intégrales.

Informations :

L'utilisation du site sur de petits écrans peut parfois poser des problèmes d'affichage. L'idéal reste d'utiliser un ordinateur pour accéder au site.
